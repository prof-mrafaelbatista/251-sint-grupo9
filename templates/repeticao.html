Estrutura de Repetição

Conceito

Permitem que um bloco de código seja executado repetidamente enquanto uma condição for verdadeira.

Aplicação

Muito usadas para percorrer coleções, contar, verificar entradas de dados, entre outros.

Exemplo de código

Usando while

contador = 1

while contador <= 5:
    print("Contagem:", contador)
    contador += 1

Usando for

for i in range(1, 6):
    print("Contagem:", i)
