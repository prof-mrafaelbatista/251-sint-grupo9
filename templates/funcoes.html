Funções e Procedimentos

Conceito

Blocos de código que realizam tarefas específicas e podem ser reutilizados. Funções retornam valores; procedimentos (em algumas linguagens) apenas executam ações.

Aplicação

Usados para modularizar o código, melhorar a organização e evitar repetição.

Exemplo de código



        # Função com retorno
def soma(a, b):
    return a + b

# Procedimento (sem retorno específico)
def saudacao(nome):
    print(f"Olá, {nome}!")

# Usando as funções
resultado = soma(5, 3)
print("Soma:", resultado)
saudacao("Maria")
