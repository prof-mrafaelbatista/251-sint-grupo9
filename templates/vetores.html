Vetores e Matrizes

Conceito

Vetor (lista): Estrutura linear que armazena múltiplos valores em uma única variável. Matriz (lista de listas): Estrutura bidimensional que armazena dados em linhas e colunas.

Aplicação

Usados para armazenar e manipular conjuntos de dados de forma organizada.

Exemplo de código

# Vetor
notas = [7.5, 8.0, 9.2]
media = sum(notas) / len(notas)
print("Média:", media)

# Matriz
matriz = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print("Elemento [1][2]:", matriz[1][2])  # Imprime 6
